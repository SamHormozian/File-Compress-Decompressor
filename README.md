<h1>File Compressor/Decompressor</h1>


<h2>Description</h2>
This Project presents a robust implementation of the Huffman coding algorithm, renowned for its effective data compression capabilities. Our implementation offers an intuitive and performant way to compress and decompress data files, showcasing the algorithm's utility in reducing file size without losing information.
<br />


<h2>Languages and Utilities Used</h2>

- C++

<h2>Environments Used </h2>

- <b>Visual Studio Code</b>
